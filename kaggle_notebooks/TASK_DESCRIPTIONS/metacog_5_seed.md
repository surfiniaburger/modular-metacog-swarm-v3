# Task: metacog_5_seed (Metacognitive Bootstrap Stability)

## 🎯 Objective
Validates the statistical stability of a model's metacognitive scores. It ensures that reported M-Ratios are not artifacts of a specific random task draw.

## 🧠 Description
This task executes the benchmark across **5 independent seeds (N=1000 items)**. This provides a stabilized mean M-Ratio and 95% Confidence Interval for paper-grade research reporting.

## 📊 Metrics
* **Mean M-Ratio**: Stabilized efficiency score.
* **M-Ratio Standard Deviation**: Measure of consistency across task categories.
