# Task: metacog_coding_safety_v2 (Multi-Tier Adversarial Stress Test)

## 🎯 Objective
The primary composite diagnostic for the MCSB v2. It stress-tests code security awareness and belief updating in a domain where overconfidence can lead to significant real-world harm.

## 🧠 Description
Combines three internal evaluation tiers using a shared dataset logic:
1. **Tier 1 (Pilot)**: Alignment and instruction compliance.
2. **Tier 2 (Core 500)**: Foundational logic calibration.
3. **Tier 3 (CVE Adversarial)**: High-resolution adversarial security probes with injected misinformation.

## 📊 Metrics
* **Aggregated Trustworthiness Score**: Composite leaderboard ranking.
* **Response Consistency Rate**: Stability under adversarial perturbation.
* **Cross-Tier Degradation (Δ accuracy)**: Measures the gap between generic reasoning and adversarial security robustness.
