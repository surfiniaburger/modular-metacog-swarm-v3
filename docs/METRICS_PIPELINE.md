# Metacognitive Metrics Pipeline: Log-to-Truth Extraction

The "In-varia" suite moves beyond aggregate calibration metrics (like ECE) by reconstructing the **verbatim trial-level reality** of model performance. This document details the automated pipeline that transforms raw `kbench` logs into truth-based reliability diagrams.

## 1. The Input: `kbench` Run Logs
The pipeline consumes standard `*.run.json` files generated by the Kaggle cookbook (`kbench`) framework. These logs contain:
- **Conversation IDs:** Format `trial_{idx}-{session_id}`.
- **Request/Response Pairs:** Nested JSON strings containing the model's `choice` and `confidence_bin` (1-6).

## 2. Reconstructing Ground Truth
Because the metacognitive benchmark uses a procedural dataset, we must ensure we are comparing model responses to the **correct** ground-truth answer for each specific trial index.

To achieve this without high-latency database lookups, the extraction script re-runs the procedural generator with the **Global Task Seed (42)**:

This ensures that `trial_0` in the model log always maps to the same ground-truth answer as the benchmark environment, maintaining scientific parity.

## 3. The Extraction Algorithm
The script utilizes a **Batch-Pass** approach to process all `.run.json` files in the workspace:

1.  **Trial Mapping:** Iterates through `conversations`, extracting the `trial_idx` from the ID.
2.  **Double-Turn Logic:**
    - **Turn 1 (Static):** Extracts the model's initial confidence before evidence pressure.
    - **Turn 2 (Dynamic):** Extracts the model's updated confidence after evidence (supportive or gaslighting) is injected.
3.  **Binning:** For each model, it creates a count-accurate histogram of `[confidence_bin] -> (accuracy_sum, total_count)`.

## 5. Dashboard Integration
The Next.js frontend merges this raw bin data into the existing dashboard results at runtime using a `useMemo` strategy. This allows the **Reliability Diagram** to plot a "True Calibration Line" vs. the "Perfect Calibration" reference, revealing overconfidence or underconfidence with 100% precision.
